# 🏦 Loan Approval Prediction System

📌 Overview
- This project is a Machine Learning-based Loan Approval System that predicts whether a loan application should be approved or rejected based on applicant details. The goal is to automate and improve the decision-making process using data-driven insights.

🚀 Features
- Predict loan approval status (Approved / Rejected)
- Data preprocessing and handling missing values
- Feature encoding for categorical variables
- Model training and evaluation
- User-friendly interface (if using Streamlit/FastAPI)

🧠 Tech Stack
- Python
- Pandas, NumPy
- Scikit-learn
- Streamlit / FastAPI (for deployment)
- Matplotlib / Seaborn (for visualization)

▶️ Installation & Setup
1. Clone the repository
git clone https://github.com/raajbbhattacharya/loan-approval-system.git
cd loan-approval-system
2. Install dependencies
pip install -r requirements.txt
3. Run the application
streamlit run app.py
